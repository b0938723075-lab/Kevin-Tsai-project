蔡康永（英語：Kevin Tsai，1962年3月1日—），臺灣男主持人、作家和導演。 蔡康永早年是影評人和編劇，編劇的電影包括《阿嬰》(1990)，《方世玉 "方世玉 (電影)")》(1993)，後成為作家和主持人。2004年至2016年，和小S搭檔主持《康熙來了》，在華人世界大獲成功，曾獲第40屆金鐘獎最佳綜藝節目主持人獎。 2014年至2020年，和馬東等人搭檔主持《奇葩說》。他還曾兩度主持金鐘獎，七度主持金馬獎。2017年，他導演的首部電影《吃吃的愛》上映。 蔡康永出版過多本著作，包括《痛快日記》、《LA流浪記》、《那些男孩教我的事》、《蔡康永的說話之道》系列和《蔡康永的情商課》系列等。 ## 生平 蔡康永生於臺灣臺北市，父母來自上海、原籍浙江省寧波市。父親蔡天鐸是律師，1949年前曾是「中聯企業公司」的股東，1949年沉沒的豪華客輪「太平輪」正是屬於該公司。蔡康永自幼兒園至就讀臺北市私立再興中學學生生涯中，多次代表學校參加過作文、辯論、演講等校際比賽。也曾經學過唱京劇。大學錄取東海大學 "東海大學 (台灣)")社會工作學系，後轉至外國語文學系。 [...] 維基百科 自由的百科全書 ## 目次 # 蔡康永 | 蔡康永 | | --- | | 攝於2020年 | | | 節目主持 | | | 英文名 | Kevin Tsai | | 出生 | (1962-03-01) 1962年3月1日（64歲） 臺灣臺北市 "臺北市 (省轄市)") | | 職業 | 主持人、作家、導演 | | 語言 | 國語、英語、吳語上海話 | | 母校 | 東海大學 "東海大學 (台灣)")（外文系學士） 加州大學洛杉磯分校（碩士） | | 父母 | 蔡天鐸（父） | | 出道作品 | 《流行都市》 | | 代表作品 | 電視節目： 《真情指數》 《康熙來了》 《奇葩說》 書籍： 《蔡康永的說話之道》系列 《蔡康永的情商課》系列 電影： 《吃吃的愛》（導演） | | 活躍年代 | 1994年至今 | | 網路電影資料庫（IMDb）資訊 | | | | | | | 獎項 | | --- | | 金鐘獎 | | | 談話節目獎 2000年 《真情指數》 娛樂綜藝節目獎 2003年 《兩代電力公司》 娛樂綜藝節目獎 2005年 《兩代電力公司》 綜藝節目主持人獎 2005年 《康熙來了》 | | | | | 獎項 | | --- | | 金鐘獎 | | | 談話節目獎 2000年 《真情指數》 娛樂綜藝節目獎 2003年 《兩代電力公司》 娛樂綜藝節目獎 2005年 《兩代電力公司》 綜藝節目主持人獎 2005年 《康熙來了》 | | 蔡康永（英語：Kevin Tsai，1962年3月1日—），臺灣男主持人、作家和導演。 [...] 1990年，蔡康永獲得美國加州大學洛杉磯分校電影電視研究所編導製作碩士學位，返回臺灣參加電影製片以及編劇、影評的工作，後跨足廣播影視圈。曾擔任臺北之音總監、時尚雜誌《GQ》中文版總編輯。1994年受傳訊電視大地頻道總監施南生提攜，在大地頻道主持《一千零一頁》與《流行都市》。1996年受張小燕 "張小燕 (臺灣)")提攜主持TVBS-G節目《翻書觸電王》，其後TVBS頻道名人訪談節目《真情指數》和TVBS-G青老年人溝通節目《兩代電力公司》獲得成功。2004年至2016年，蔡康永與臺灣女藝人徐熙娣（小S）共同主持中天電視綜藝訪談節目《康熙來了》，在華人世界廣受歡迎。2017年，蔡康永導演了首部電影《吃吃的愛》，但票房失利。 ## 個人生活 蔡康永是臺灣公眾人物中少數公開出櫃的同性戀者。2001年11月21日，蔡康永參加《文茜小妹大》節目的錄影，代班主持人李敖問他：「蔡康永，你是不是gay？」蔡康永在節目中坦誠以告，公開出櫃。 蔡康永的伴侶是劉坤龍（George），兩人在1994年相識相戀，日後數度傳出分手的傳聞。 ## 爭議 蔡康永不時遭到抵制和審查，部分與其公開的同志身份有關。2006年，演員王羽表示自己拒絕蔡康永的節目邀請，因為「對他的性向不欣賞。」 2009年，湖南衛視《背後的故事》蔡康永專訪在錄製完成，播出預告後被緊急封殺。2012年-2013年，東方衛視《頂級廚師》第二季、《媽媽咪呀》刪除蔡康永畫面。 2015年，蔡康永在愛奇藝《奇葩說》討論出櫃時，落淚稱「我們不是妖怪」，遭廣電總局以「挑戰傳統道德與價值觀」為由要求下架。 2017年，蔡康永、小S宣傳電影《吃吃的愛》期間試圖錄製湖南衛視《快樂大本營》，但節目報批未過。2025年，《喜劇之王單口季第二季》刪除蔡康永畫面。 ## 主持作品 ### 電視節目 （有特別註明日期者，皆指任期。）
